## 📌 О проекте

**ReadyPro** — это универсальная платформа для организации и проведения массовых мероприятий: форумов, конференций, хакатонов. Мы объединяем все этапы жизни мероприятия в одном месте: от планирования и назначения задач до интерактивного взаимодействия участников во время события.

Платформа автоматизирует рутину, избавляет от хаоса в мессенджерах и таблицах, предоставляя удобные инструменты для всех ролей: организаторов, кураторов секций, спикеров и участников.

---

## 🔥 Проблема

Сейчас организаторы мероприятий вынуждены использовать зоопарк инструментов:
- 📄 Сайт с регистрацией, который наполняется вручную;
- 📑 Программа мероприятия в отдельном PDF-файле;
- 💬 Мессенджеры для коммуникации и обмена файлами;
- 📊 Таблицы для отслеживания задач и готовности секций.

Это приводит к путанице, потере информации и низкой вовлечённости участников.

---

## ✅ Наше решение

**ReadyPro** предоставляет единое цифровое пространство для всех этапов:

### До мероприятия
- 📋 **Конструктор структуры** – создание секций, докладов, кейсов в несколько кликов.
- 👥 **Назначение ответственных** – организаторы, кураторы секций, спикеры.
- 📊 **Дашборд готовности** – визуализация прогресса наполнения секций, дедлайны по задачам.
- 🔔 **Умные уведомления** – напоминания о приближающихся задачах.
- 📅 **Автоматическое формирование программы** – программа генерируется на основе внесённых данных и доступна онлайн.

### Во время мероприятия
- 🎫 **Личный кабинет участника** – выбор докладов, построение личного расписания.
- 💬 **Чат внутри секции** – вопросы спикерам, обсуждения с другими участниками.
- 📝 **Обратная связь и анкеты** – оценка докладов, заполнение форм, опросы.
- 📈 **Инструменты для кураторов** – модерация вопросов, заметки по докладам, быстрый сбор отчёта по секции.

### После мероприятия
- 📊 **Аналитика** – статистика посещений, оценки, отзывы.
- 📄 **Генерация итоговых отчётов** – по секциям, спикерам, докладам.
- 🏆 **Рейтинги и достижения** – лучшие доклады, активные участники.

## 🛠️ Технологический стек

- **Клиент:** Flutter (Dart 3.11+), Material Design
- **Бэкенд и данные:** [Supabase](https://supabase.com/) — аутентификация, REST API (PostgREST), хранилище файлов
- **База данных:** PostgreSQL (управляется Supabase)
- **Реальное время:** Supabase Realtime (потоки изменений таблиц, в т.ч. для сообщений в чатах)
- **Архитектура приложения:** внедрение зависимостей ([get_it](https://pub.dev/packages/get_it)), репозитории поверх Supabase SDK
- **Тестирование:** `flutter_test`, [mocktail](https://pub.dev/packages/mocktail)
